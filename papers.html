Two papers, from two different lives.

DerainGAN: Single Image Deraining Using Wasserstein GAN

Sahil Yadav, Aryan Mehra, Honnesh Rohmetra, Rahul Ratnakumar, Pratik Narang. Multimedia Tools and Applications (Springer), 2021.

Rain is a nuisance for computer vision: streaks and droplets sit on top of an image and confuse anything trying to read it. This paper trains a Wasserstein GAN to take a single rainy photograph and return a clean one, learning to pull the rain off the scene behind it. It was my first paper, written as an undergraduate at BITS Pilani.

IACT event analysis with the MAGIC telescopes using deep convolutional neural networks with CTLearn

T. Miener, R. López-Coto, J. L. Contreras, J. G. Green, D. Green for the MAGIC Collaboration, E. Mariotti, D. Nieto, L. Romanato, S. Yadav. ADASS XXXI, 2021.

A report from the CTLearn collaboration on using deep convolutional networks to analyse real data from the MAGIC gamma-ray telescopes: separating gamma-ray events from the background and reconstructing their properties, the work that turns telescope images into physics.
